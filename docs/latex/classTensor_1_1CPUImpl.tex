\doxysubsection{Detailed Description}
CPU implementation of \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}, backed by std\+::vector$<$float$>$.

All operations iterate over the data using \doxylink{structTensorLayout}{Tensor\+Layout} descriptors. The caller is responsible for layout validity, see \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}.

Overridden methods follow the same semantics documented in \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}.
