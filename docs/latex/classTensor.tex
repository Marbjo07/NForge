\doxysubsection{Detailed Description}
Multi-\/dimensional array with backend specific implementation.

A {\ttfamily \doxylink{classTensor}{Tensor}} owns a contiguous block of float data managed by a backend specific {\ttfamily \doxylink{classTensor_1_1Impl}{Impl}} object. Elements can be accessed through {\ttfamily \doxylink{classTensor_1_1View}{Tensor\+::\+View}}, which describes a sub region via offset, shape, and stride layout.
