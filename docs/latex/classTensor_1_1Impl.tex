\doxysubsection{Detailed Description}
Abstract interface for backend specific tensor storage and operations.

All data access goes through \doxylink{structTensorLayout}{Tensor\+Layout} descriptors, not raw indices.

The caller must ensure layouts are valid, since the implementation trusts that {\ttfamily rhs\+Layout} is correct for {\ttfamily rhs\+Impl} and {\ttfamily lhs\+Layout} is correct for its own memory.

The caller is responsible for broadcasting.
