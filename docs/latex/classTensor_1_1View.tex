\doxysubsection{Detailed Description}
Non-\/owning reference to a sub region of a \doxylink{classTensor}{Tensor}.

A \doxylink{classTensor_1_1View}{View} describes a sub region of a \doxylink{classTensor}{Tensor} via an position, shape, and stride layout. Multiple views can reference the same underlying data at different offsets and shapes.
