\doxysubsection{Detailed Description}
CUDA implementation of \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}, backed by device memory.

All operations launch CUDA kernels using \doxylink{structTensorLayout}{Tensor\+Layout} descriptors. The caller is responsible for layout validity, see \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}.

Overridden methods follow the same semantics documented in \doxylink{classTensor_1_1Impl}{Tensor\+::\+Impl}.
