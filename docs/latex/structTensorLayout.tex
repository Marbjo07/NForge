\doxysubsection{Detailed Description}
Describes the memory layout of a tensor. 

A tensor layout stores\+:
\begin{DoxyItemize}
\item the tensor shape,
\item the stride for each dimension,
\item an optional storage offset,
\item the number of active dimensions ({\ttfamily rank}).
\end{DoxyItemize}

\begin{DoxyInvariant}{Invariant}
{\ttfamily 0 \texorpdfstring{$<$}{<}= rank \&\& rank \texorpdfstring{$<$}{<}= MAX\+\_\+\+DIMS} 

Elements in {\ttfamily shape} and {\ttfamily strides} are only meaningful for the first {\ttfamily rank} active entries 
\end{DoxyInvariant}
